Устанавливаем AutoIt на ПК
Запускаем встроенный редактор SciTE.exe
Запускаем Au3Info_x64.exe
С помощью Au3Info собираем данные по необходимым Windows окнам\кнопкам
Пишим код в SciTE (см. autoit.au3 файл)
где "Открытие" - это тайтл
"Edit1" - это Class + Instance из окна Basic Control Info
Сохраняем и правым кликом мыши конвертируем в script.exe
Его и запускаем через команду Runtime в тесте
